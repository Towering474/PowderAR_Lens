# Spice My Rice – Mobile AR Lens Studio App

Mobile Augmented Reality lens for visualization of food decoration patterns.

---

## Overview

This repository contains the **Mobile AR prototype** lens developed for the *Spice My Rice* research project. These repository contains the studio lens project that implements the AR system, the application that makes use of it is PowderAR_App. The system explores how AR visualization can support **food decoration tasks** by projecting decorative patterns directly onto the food surface.

The prototype allows users to preview and manipulate decorative patterns in AR before executing them manually using traditional ingredients such as cinnamon or whipped cream. The goal is to support both **conceptualization and execution** of decorative designs by providing spatial visual guidance.

This prototype corresponds to the **mobile AR implementation described in the paper**.

## Features

- AR pattern visualization through marker tracking
- 2D UI interaction for pattern height, opacity and design
